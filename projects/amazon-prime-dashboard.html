An interactive Tableau dashboard dissecting the Amazon Prime Video catalog — mapping content trends, genre distribution, ratings, and release patterns across thousands of titles. Features custom radial bar charts and a branded dark design system inspired by the platform.

Methodology

How it was built

The dataset contains metadata for over 9,000 Amazon Prime Video titles sourced from Kaggle. Data cleaning addressed missing ratings, inconsistent genre labels, and date formatting. In Tableau, custom calculated fields powered the radial bar chart layout, while parameter actions enable cross-sheet filtering — clicking any genre or year updates the entire dashboard in sync. The color system was derived directly from Amazon Prime's brand palette.
