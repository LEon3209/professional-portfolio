# Instagram Takeover assets

- `stories/story-01.jpg` … `story-12.jpg` — story screenshots shown in the
  phone-frame gallery (`.jpeg` also works; numbering is two digits, gaps fine).
- The Instagram posts are embedded directly in the page HTML
  (`projects/unige-instagram-takeover.html`, "The Posts" section) using
  Instagram's official embed code — to add another, paste its blockquote
  inside the `instagram-grid` div.
- `certificate.pdf` is a copy of the certificate from `certs/`; the page embeds it.
