Genre-Based Music
Recommendation System

Jul 2024 · Bachelor's Thesis · LUISS Guido Carli

My bachelor's thesis: a genre-based music recommender built on Spotify's Web API. It draws on users' listening history and stated preferences to generate personalized recommendations, and closes the loop with a survey evaluating how well those recommendations actually land with listeners.

At a Glance

How the recommender works

Data Source
Spotify Web API
Listening history, saved tracks, and audio features pulled per user through the official API.
Approach
Genre-Based
Recommendations are generated from the genre profile of each user's actual listening behaviour.
Evaluation
User Survey
Real users rated the quality of their recommendations, providing an empirical quality check.
